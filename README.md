# Azure Cloud Project

## Overview
This project demonstrates deploying a cloud-based architecture on Microsoft Azure.

## Architecture
The solution includes:
- Azure Load Balancer
- Two Virtual Machines (Web Servers)
- Azure SQL Database
- Azure Key Vault

## Components
- VM1 and VM2 host web applications
- Load Balancer distributes incoming traffic
- SQL Database stores application data
- Key Vault secures sensitive information

## Deployment Steps
1. Created Virtual Machines
2. Configured Load Balancer
3. Deployed Azure SQL Database
4. Stored secrets in Key Vault
5. Configured monitoring and alerts
